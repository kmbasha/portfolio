Multi-Brand Retail Integration

250+ Locations across UAE & GCC Region

Client Jashanmal Group, Burberry
Duration 2014-2017
Budget $800K
Role Retail Systems Integration Manager

Program Overview

Led centralized time & attendance integration program connecting 250+ retail locations across UAE and GCC region for luxury retail brands including Jashanmal Group and Burberry.

Technical Implementation

Custom Integration Development

  • Custom WebAPI development for multi-vendor device connectivity
  • Developed abstraction layer supporting 10+ T&A device brands (ZKTeco, Suprema, Anviz, eSSL, etc.)
  • Standardized data format across disparate systems
  • RESTful API for Oracle ERP integration

Data Synchronization

  • Real-time data synchronization using CI pipelines
  • Automated shift calculations and overtime computation
  • Exception handling and conflict resolution
  • Bi-directional sync (attendance to ERP, employee master data from ERP)

Centralized Management

  • Single management console for 250+ locations
  • Real-time attendance visibility across all stores
  • Automated reporting and analytics
  • Workforce planning dashboards
  • Mobile app for managers
