IBM Legacy System Modernization

COBOL ERP Hardware Migration - Power Systems Upgrade
IBM AS/400 Power Systems COBOL ERP IBM FlashSystem Legacy Modernization

Project Overview

Client: Manufacturing/Distribution enterprise | Duration: 2023 | Budget: $250K

Managed critical IBM AS/400 (iSeries) hardware upgrade supporting legacy COBOL ERP. Coordinated with IBM, storage vendor, application team, business users.

Project Scope - Hardware Infrastructure Migration

Important: This was a hardware infrastructure project, NOT an application migration:

  • PM Role: Infrastructure project management, vendor coordination, testing oversight
  • Application Handling: All COBOL application changes managed by application vendor
  • Objective: Upgrade underlying hardware platform while preserving existing COBOL ERP applications

Technical Scope

IBM Power Systems Upgrade

  • Hardware: Power9 server replacing aging Power7 system
  • Processor & Memory: Increased capacity for improved performance
  • OS Upgrade: IBM i 7.4 (latest supported version)
  • Backwards compatibility: Maintained support for existing COBOL applications

Storage Migration

  • IBM FlashSystem: All-flash storage replacing traditional spinning disks
  • SAN fabric upgrade: 16Gbps Fibre Channel for higher throughput
  • Snapshot/replication: Enhanced data protection capabilities
  • Performance improvement: Significantly improved IOPS and reduced latency

COBOL ERP Compatibility Testing

  • Application compatibility: Validated COBOL programs on new hardware
  • COBOL program validation: Ensured all legacy code runs correctly
  • Database integrity: Verified DB2/400 database after migration
  • Interface testing: Tested integrations with other systems
  • UAT with business users: End-to-end business process validation

Backup/DR Modernization

  • IBM Spectrum Protect: Modern backup solution replacing legacy tape backup
  • LTO-8 tape library: High-capacity tape library for long-term retention
  • DR testing procedures: Documented and tested disaster recovery
  • Retention compliance: Meeting regulatory requirements

Implementation Approach

  • Phased approach: Test environment first, then production
  • Comprehensive rollback procedures: Ready if issues encountered
  • 24/7 support: During transition period
  • Weekend cutover: Minimizing business disruption
  • Hypercare period: 2 weeks of intensive monitoring post-migration

Stakeholder Coordination

  • IBM: Hardware delivery, installation, configuration
  • Storage vendor: FlashSystem deployment and integration
  • Application team: COBOL compatibility validation
  • Business users: UAT and process validation
  • Management: Weekly status updates and risk reporting

Outcomes & Benefits

  • 60% performance improvement in ERP transactions
  • Zero ERP downtime during upgrade
  • Maintained legacy COBOL compatibility - all applications running correctly
  • Provided runway for gradual cloud modernization (future state)
  • Improved backup/recovery with modern tooling
  • Met compliance requirements for data retention
  • Extended platform lifespan 5+ years
